SECURITY TOKEN STANDARD

What is ST-20?

ST-20 is a free, open standard that describes how to build transfer restricted securities tokens on the Ethereum blockchain. While most tokens are have no limitations on who can receive, or participate in the token sale, ST-20 restricts token transfers to verified investors only.

Security Token Standard started as a EIP draft written by @frozeman, @adamdossa, @pabloruiz55, and more at the Security Token Roundtable

This Security Token Standard has since moved out of beta and into a community formalized v1 spec, supported and endorsed by a large number of projects from across the crypto ecosystem. This EIP wouldn't be possible without time and dedication from a number of community members.
